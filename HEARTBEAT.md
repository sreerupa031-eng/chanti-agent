# Heartbeat

## Daily Tasks

### 9:00 AM — Morning Briefing
- Check Google Calendar for today's events and schedule
- Summarize top 5 unread emails in bullet points
- Share any important reminders or deadlines
- Send everything to Telegram in a warm friendly tone
- Start message with "Good morning Rupa! 🌟 Here's your day:"

### 6:00 PM — Evening Study Reminder
- Send a friendly study reminder to Telegram
- Check if any tasks from the morning are still pending
- Encourage Rupa with a motivational message
- Start message with "Hey Rupa! 📚 Time to study!"

## Weekly Tasks

### Every Sunday 8:00 PM — Weekly Preview
- Check Google Calendar for the upcoming week
- Send a summary of what's coming up this week to Telegram
- Start message with "Hey Rupa! 🗓️ Here's your week ahead:"

## Important Rules for Heartbeat
- Keep all messages short and friendly
- Never send long walls of text
- Always use Telegram to deliver messages
- If a task fails, try once more then notify Rupa
- Keep heartbeat tasks simple — spawn subagent for complex work
